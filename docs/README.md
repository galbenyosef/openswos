# docs/

Design notes, reverse-engineered asset format documentation, and architectural decision records.

- `formats/` — what we've learned about each SWOS file format (TEAM.DAT, PITCH.*, SPRITE.DAT, .EDT, .TMD, .RAW, etc.).
- `decisions/` — Architecture Decision Records (ADRs) for locked calls (engine, language, licence, layout, netcode model when picked, etc.).

For the high-level project brief, see `../CLAUDE.md`.
